# Plantilla: Detectar contradicciones

**Cuaderno destino:** cualquiera
**Uso:** Identificar inconsistencias entre documentos cargados

---

## Consulta

En las fuentes cargadas sobre {{TEMA}}, identificar:

1. Afirmaciones contradictorias entre documentos (citar ambas fuentes).
2. Conceptos definidos de forma diferente en distintos documentos.
3. Reglas de negocio que se aplican de forma inconsistente.
4. Requisitos que se excluyen mutuamente.

**Instrucciones:**
- Listar cada contradicción con: fuente A (cita), fuente B (cita), naturaleza del conflicto.
- Máximo 15 contradicciones.
- Ordenar de mayor a menor impacto potencial.
- Si no se encuentran contradicciones, indicarlo explícitamente.
- No resolver las contradicciones — solo identificarlas para revisión humana.
