# Plantilla: Relacionar actor, proceso y módulo

**Cuaderno destino:** actores-y-permisos | gestión-académica
**Uso:** Trazar la relación entre un actor, sus procesos y los módulos involucrados

---

## Consulta

Para el actor {{NOMBRE_ACTOR}} en EDURA:

1. ¿Qué procesos puede iniciar o en cuáles participa?
2. ¿A qué módulos del sistema accede y con qué permisos?
3. ¿Qué información visualiza y qué información puede modificar?
4. ¿Qué restricciones institucionales o de rol aplican?
5. ¿Hay flujos donde este actor interactúa con otros actores?

**Instrucciones:**
- Máximo 250 palabras.
- Citar fuente para cada permiso o restricción mencionada.
- Indicar vacíos: qué procesos del actor NO están documentados.
- No asumir permisos no documentados explícitamente.
