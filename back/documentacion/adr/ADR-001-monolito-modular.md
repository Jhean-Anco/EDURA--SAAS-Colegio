# ADR-001 Monolito modular

- Estado: aceptado.
- Contexto: EDURA inicia con alcance acotado y crecimiento progresivo.
- Decisión: organizar el backend como monolito modular.
- Alternativas: microservicios, modularidad plana.
- Consecuencias: menor complejidad operativa inicial.
- Riesgos: acoplamiento si no se respetan fronteras.
- Fecha: 2026-06-19.
